# Stroke Prediction Analysis

## Project
This project predicts the risk of **Stroke** using patient data. It helps identify key risk factors and provides predictions to support medical decisions.  

## Dataset
- **5,110 patients** with 12 variables  
- **Demographics:** Age, gender, residence, ever_married 
- **Health:** Hypertension, heart disease, BMI  
- **Lifestyle:** Smoking, work type  
- **Target:** `stroke` (1 = yes, 0 = no)  

> Note: Only **4.9%** of patients had a stroke, so we handled class imbalance carefully.

##  Tools
- Python( Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn) 

##  Work Done
- Cleaned and preprocessed data  
- Encoded categorical variables  
- Scaled features for better performance   
- Tested models: Logistic Regression, Random Forest and Support Vector Machine  
- Selected Logistic Regression as the final model
